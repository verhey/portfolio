# My resume/CV

For the past few years I've worked in data engineering, though my first jobs in tech were in full-stack web development in PHP/Jquery/Wordpress/other monstrosities. I've done a lot of things, though you probably don't want me to write PHP or Wordpress anymore. If you'd like to get in touch, see the [about page](/about) on this site.

* 🏴 **[LaunchDarkly](https://launchdarkly.com/)** (October 2021 -> Current)
  * I was an early hire on LaunchDarkly's internal data team, helped grow that team from 3 to 9. I now work on the product team building our core data platform and experimentation products.
* ✈️ **[Yapta (acquired by Coupa Software)](https://www.geekwire.com/2020/coupa-software-acquiring-seattle-startup-yapta-help-businesses-cut-travel-costs/)** (November 2019 ->  October 2021)
