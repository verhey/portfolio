# portfolio

My personal portfolio website built with Eleventy.

## Prerequisites

- Node.js (version 16 or higher)
- npm

## Installation

1. Clone this repository
2. Install dependencies:
   ```
   npm install
   ```

## Running the project

### Development mode

To run the site with live reloading:

```
npm start
```

This will start a local server at http://localhost:8080

### Build for production

To build the site for production:

```
npm run build
```

This creates optimized static files in the `build` directory.

## Project structure

- `src/`: Source content (Markdown files, templates)
- `public/`: Static assets
- `.eleventy.js`: Eleventy configuration
- `build/`: Output directory (generated files)
